Arduino Simple Robot
====================

Code for a simple Arduino based bot using the following components:

  DFRobot Pirate Chassis
  Arduino Mega 2560 V3
  L298N Motor Shield
  HC-SR04 Ultrasonic Sensor
  Ultrasonic Sensor
  Standard 16x2 Character LCD Screen
  Infra-red Sensors
  MMA7361 Three Axis Accelerometer

See www.kevinpeat.com for more information.
